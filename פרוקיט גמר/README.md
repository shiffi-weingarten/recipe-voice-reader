# 🍳 ספר המתכונים שלי - פרויקט גמר

## תיאור הפרויקט
אתר מתכונים מתקדם עם פונקציית הקראה קולית, המאפשר ניהול מתכונים אישיים והקראה שלב אחר שלב תוך כדי בישול.

## תכונות עיקריות

### ✅ מסכים
- **מסך התחברות** - כניסה מאובטחת עם שם משתמש וסיסמה
- **מסך הגדרות** - התאמה אישית של חוויית המשתמש
- **מסך רשימת מתכונים** - תצוגה, חיפוש והוספת מתכונים
- **מסך תצוגת מתכון** - הצגת מתכון עם הקראה קולית

### 🎯 פונקציונליות
- **הקראה קולית חכמה** - הקראת המרכיבים והוראות ההכנה שלב אחר שלב
- **התאמה אישית מלאה**:
  - זמן המתנה בין שורות (0-10 שניות)
  - מהירות הקראה (0.5x-2x)
  - גובה קול
  - מצב כהה/בהיר
  - גודל גופן (קטן/בינוני/גדול)
- **ניהול מתכונים**:
  - הוספת מתכונים חדשים
  - מחיקת מתכונים
  - חיפוש מתכונים
- **שמירת נתונים** - כל הנתונים נשמרים ב-LocalStorage

### 👥 כניסה לאתר
- **כניסה פשוטה** - הזן שם משתמש וסיסמה
- **אם המשתמש לא קיים** - המערכת תיצור אותו אוטומטית!
- **אם המשתמש קיים** - תיכנס עם הסיסמה שלך

## טכנולוגיות
- **HTML5** - מבנה סמנטי ונקי
- **CSS3** - עיצוב מודרני עם CSS Variables ואנימציות
- **JavaScript (Vanilla)** - קוד נקי ויעיל ללא תלויות חיצוניות
- **Web Speech API** - הקראה קולית מתקדמת
- **LocalStorage API** - שמירת נתונים מקומית

### 💾 מה זה LocalStorage?
LocalStorage זה אחסון מקומי בדפדפן שלך:
- **כל הנתונים נשמרים במחשב שלך** (לא בשרת)
- **המשתמשים, המתכונים וההגדרות שלך נשמרים אצלך**
- **הנתונים נשארים גם אחרי סגירת הדפדפן**
- **כל משתמש שנכנס לאתר מקבל את המתכונים ברירת המחדל ויכול להוסיף משלו**
- **אם תנקה את ה-cache של הדפדפן, הנתונים ימחקו**

## מבנה הפרויקט
```
פרוקיט גמר/
├── index.html      # קובץ HTML ראשי
├── styles.css      # עיצוב מלא של האתר

```

## הרצה מקומית
1. פתח את הקובץ `index.html` בדפדפן
2. התחבר עם אחד המשתמשים הקיימים
3. התחל להשתמש באתר!

## שימוש באתר

### כניסה
1. הזן שם משתמש וסיסמה (כל שם שתבחר)
2. אם זה פעם ראשונה - המערכת תיצור לך משתמש חדש
3. אם כבר יש לך משתמש - תיכנס עם הסיסמה שלך

### צפייה במתכונים
1. לאחר התחברות תראה את רשימת המתכונים
2. השתמש בשורת החיפוש למציאת מתכון ספציפי
3. לחץ על כרטיס מתכון לצפייה מלאה

### הוספת מתכון
1. לחץ על "➕ הוסף מתכון"
2. מלא את שם המתכון
3. הזן מרכיבים (כל מרכיב בשורה נפרדת)
4. הזן הוראות הכנה (כל שלב בשורה נפרדת)
5. לחץ על "הוסף מתכון"

### הקראה קולית
1. פתח מתכון
2. לחץ על "▶️ התחל הקראה"
3. **המערכת תקריא:**
   - תחילה את כל המרכיבים
   - ואז את הוראות ההכנה שלב אחר שלב
4. השתמש ב-"⏸️ השהה" להשהיה
5. השתמש ב-"⏹️ עצור" לעצירה מלאה

### הגדרות אישיות
1. לחץ על "⚙️ הגדרות" בראש העמוד
2. התאם את ההגדרות לפי העדפתך
3. לחץ על "שמור הגדרות"

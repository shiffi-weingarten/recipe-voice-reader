# 🌐 הוראות העלאה לאינטרנט - חינם!

## אופציה 1: Netlify (הכי קל - 2 דקות!)

### שלבים:
1. **היכנסי ל-https://www.netlify.com**
2. **לחצי על "Sign up" ותירשמי (חינם לגמרי)**
3. **גררי את כל התיקייה "פרוקיט גמר" לאזור ה-Drop**
4. **זהו! תקבלי קישור מיידי!**

### הקישור יהיה בפורמט:
```
https://your-site-name.netlify.app
```

---

## אופציה 2: GitHub Pages (מקצועי יותר)

### שלבים:
1. **צרי חשבון ב-GitHub.com** (אם אין לך)
2. **צרי Repository חדש:**
   - לחצי על "New repository"
   - תני שם: `recipe-book`
   - סמני "Public"
   - לחצי "Create repository"

3. **העלי את הקבצים:**
   - לחצי על "uploading an existing file"
   - גררי את הקבצים: `index.html`, `styles.css`, `app.js`, `README.md`, `סקיצה.html`
   - לחצי "Commit changes"

4. **הפעילי GitHub Pages:**
   - לכי ל-Settings > Pages
   - תחת "Source" בחרי: `main` branch
   - לחצי "Save"
   - **חכי 2-3 דקות**

5. **הקישור שלך:**
```
https://YOUR-USERNAME.github.io/recipe-book
```

---

## אופציה 3: Vercel (מהיר מאוד)

### שלבים:
1. **היכנסי ל-https://vercel.com**
2. **לחצי "Sign Up" והירשמי עם GitHub**
3. **לחצי "New Project"**
4. **גררי את התיקייה**
5. **לחצי "Deploy"**

### הקישור יהיה:
```
https://your-project.vercel.app
```
